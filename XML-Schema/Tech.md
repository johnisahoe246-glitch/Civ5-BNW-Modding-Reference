The Tech value is the list of technologies that the team begins with. By defining a list of technologies here, you control how much prior knowledge each team will begin with.


EG: Tech=TECH_MYSTICISM will give the tech mysticism to the team. The values are defined in the file "CIV4TechInfos.xml" (found in your Civilization 4 directory\Assets\XML\Technologies).
