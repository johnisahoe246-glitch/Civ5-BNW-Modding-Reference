The StartingEra value defines the era that the civilization begins the game in regards to graphics. These values are defined in the file "CIV4EraInfos.xml" (found in your Civilization 4 directory\Assets\XML\GameInfo).  The possible values are:

* ERA_ANCIENT
* ERA_CLASSICAL
* ERA_MEDIEVAL
* ERA_RENAISSANCE
* ERA_INDUSTRIAL
* ERA_MODERN
* ERA_FUTURE


EG: StartingEra=ERA_MIDEVAL
