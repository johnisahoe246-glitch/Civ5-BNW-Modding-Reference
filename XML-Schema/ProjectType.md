ProjectType defines the projects that exist in the team. This way you can define if a team project exists in a team at the start of the game/scenario. These values are defined in the file "CIV4ProjectInfo.xml" (found in your Civilization 4 directory\Assets\XML\GameInfo).  The possible values are:

* PROJECT_MANHATTAN_PROJECT
* PROJECT_THE_INTERNET
* PROJECT_SDI
* PROJECT_APOLLO_PROGRAM
* PROJECT_SS_CASING
* PROJECT_SS_THRUSTERS
* PROJECT_SS_ENGINE
* PROJECT_SS_DOCKING_BAY
* PROJECT_SS_COCKPIT
* PROJECT_SS_LIFE_SUPPORT
* PROJECT_SS_STASIS_CHAMBER
