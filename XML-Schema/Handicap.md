Handicap is the default handicap that the AI takes if no human takes this civilization. These values are defined in the file "CIV4HandicapInfo.xml" (found in your Civilization 4 directory\Assets\XML\GameInfo).  The possible values are:

* HANDICAP_SETTLER
* HANDICAP_CHIEFTAIN
* HANDICAP_WARLORD
* HANDICAP_NOBLE
* HANDICAP_PRINCE
* HANDICAP_MONARCH
* HANDICAP_EMPEROR
* HANDICAP_IMMORTAL
* HANDICAP_DEITY


EG: Handicap=HANDICAP_NOBLE
