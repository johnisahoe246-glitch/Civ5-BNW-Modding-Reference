== General ==
These two values, when used together, force the civilization to that civic at the start of the game/scenario.

== CivicOption ==
E.G. CivicOption=XXXX where XXXX is the civic category. Civic options are defined in the file "CIV4CivicOptionInfos.xml" (found in your Civilization 4 directory\Assets\XML\GameInfo).  The possible values are:

* CIVICOPTION_GOVERNMENT
* CIVICOPTION_LEGAL
* CIVICOPTION_LABOR
* CIVICOPTION_ECONOMY
* CIVICOPTION_RELIGION


EG: CivicOption=CIVICOPTION_GOVERNMENT

== Civic ==
E.G. Civic=YYYY where YYYY is the actual civic. Civics are defined in the file "CIV4CivicInfos.xml" (found in your Civilization 4 directory\Assets\XML\GameInfo).  The possible values are:

'''Government:'''
* CIVIC_DESPOTISM
* CIVIC_HEREDITARY_RULE
* CIVIC_REPRESENTATION
* CIVIC_POLICE_STATE
* CIVIC_UNIVERSAL_SUFFRAGE

'''Legal:'''
* CIVIC_BARBARISM
* CIVIC_VASSALAGE
* CIVIC_BUREAUCRACY
* CIVIC_NATIONHOOD
* CIVIC_FREE_SPEECH

'''Labor:'''
* CIVIC_TRIBALISM
* CIVIC_SLAVERY
* CIVIC_SERFDOM
* CIVIC_CASTE_SYSTEM
* CIVIC_EMANCIPATION

'''Economy:'''
* CIVIC_DECENTRALIZATION
* CIVIC_MERCANTILISM
* CIVIC_FREE_MARKET
* CIVIC_STATE_PROPERTY
* CIVIC_ENVIRONMENTALISM

'''Religion:'''
* CIVIC_PAGANISM
* CIVIC_ORGANIZED_RELIGION
* CIVIC_THEOCRACY
* CIVIC_PACIFISM
* CIVIC_FREE_RELIGION


EG: Civic=CIVIC_UNIVERSAL_SUFFRAGE
