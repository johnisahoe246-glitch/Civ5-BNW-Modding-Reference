The Speed value is the speed of the game. This is where you can set if the game is normal speed, epic speed, etc. as defined in the file "CIV4GameSpeedInfo.xml" (found in your Civilization 4 directory\Assets\XML\GameInfo). The possible values are:

* GAMESPEED_MARATHON
* GAMESPEED_EPIC
* GAMESPEED_NORMAL
* GAMESPEED_QUICK


The default value is none.
