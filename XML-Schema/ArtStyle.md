ArtStyle is the style of art that the civilization uses. This value defines building graphics and tile improvements. These values are defined in the file "GlobalTypes.xml" (found in your Civilization 4 directory\Assets\XML).  The possible values are:

* ARTSTYLE_EUROPEAN
* ARTSTYLE_ASIAN
* ARTSTYLE_SOUTH_AMERICA
* ARTSTYLE_MIDDLE_EAST
* ARTSTYLE_GRECO_ROMAN
* ARTSTYLE_BARBARIAN


EG: ArtStyle=ARTSTYLE_ASIAN
